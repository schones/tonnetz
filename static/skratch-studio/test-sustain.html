Sustain Pedal Test

How to test sustain for each instrument:
  1. Click Sustain ON (simulates pressing spacebar)
  2. Click Note On (C4) (simulates pressing a piano key)
  3. Click Note Off (C4) (simulates releasing the piano key)
  4. Verify: The note should STILL be playing after Note Off!
  5. Click Sustain OFF — now the note should fade out

If the note cuts off at step 3, sustain is broken for that instrument.
