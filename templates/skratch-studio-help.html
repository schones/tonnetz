🎵 What is Skratch Studio?

Skratch Studio is a creative playground where you can make music and art at the same time using real code — but instead of typing, you snap colorful blocks together like puzzle pieces!

🧩 Block Editor

Drag-and-drop visual programming. Your blocks turn into real JavaScript code!

🖼️ Canvas

A drawing area where your code creates shapes, animations, and visual effects.

🎹 Keyboard

A two-octave piano you can play with mouse clicks or your computer keyboard.

🎛️ Loop Pedal

Record what you play on the keyboard and loop it! Layer two recordings on top of each other.

💡 The coolest part: You can connect music to visuals! Play a note and a circle grows. Set the background to change with your pitch. Your blocks and your keyboard all work together.

🧩 How the Block Editor Works

The big area on the left is your Block Editor. On the left side of that area you'll see the toolbox — categories full of blocks to choose from.

  • Click a category name to see its blocks.
  • Drag a block into the workspace to use it.
  • Snap blocks together — they click into place like puzzle pieces.
  • Blocks with a rounded top (hat blocks 🎩) are event blocks — they decide when your code starts running.
  • Right-click a block to copy, delete, or get help.
🖥️ See the Code panel (bottom right) shows the real JavaScript that your blocks create. You can click Copy to copy it! Press ▶ Play to run your program, and ■ Stop to stop it.

🎨 Visual Blocks

Visual blocks draw things on the canvas. The canvas redraws itself 60 times per second, so your code can create smooth animations!

🎨 Visuals — Drawing shapes
  • Draw Circle Draw a circle at position (x, y) with a given size.
  • Draw Rectangle Draw a rectangle at (x, y) with width and height.
  • Draw Star Draw a 5-pointed star at (x, y) with a given size.
  • Draw Line Draw a straight line between two points.
  • Set Fill Pick the color to fill the inside of shapes.
  • Set Stroke Pick the outline color for shapes.
  • No Fill Make shapes hollow — outline only, no inside color.
  • Stroke Weight How thick the outline is (0–50 pixels).
  • Set Background Fill the whole canvas with a single color.
  • Clear Canvas Erase everything on the canvas.
  • Draw Trail Instead of clearing each frame, shapes leave a slowly fading trail. Great for psychedelic effects!
💡 Tip: The canvas is 400×400 pixels. Position (0, 0) is the top-left corner. Position (200, 200) is the center.
📐 Motion — Moving and rotating
  • Move To Jump the drawing cursor to any (x, y) position on the canvas.
  • Move to Center Jump directly to the center of the canvas (200, 200).
  • Rotate By Spin the entire drawing system by a number of degrees (–360 to +360). Everything you draw after this will be rotated!
  • Grow By Scale the drawing system up by a percentage — makes the next shapes bigger.
  • Shrink By Scale the drawing system down by a percentage (0–99%).
  • Save Position Save your current drawing position, rotation, and colors.
  • Restore Position Snap back to the last saved position and settings.
💡 Tip: Always use Save Position before rotating or moving, then Restore Position when done — that way other parts of your code aren't affected.
🔢 Math — Numbers and calculations
  • Map Value Re-scale a number from one range to another. For example: map pitch (200–800 Hz) to circle size (10–200 pixels).
  • Random Number Pick a random number between two values. Great for randomized art!
  • Canvas Width Returns 400 — the width of the canvas in pixels.
  • Canvas Height Returns 400 — the height of the canvas in pixels.
  • Frame Count A number that increases by 1 every frame (60× per second). Perfect for smooth animations over time.
  • Arithmetic Add (+), subtract (−), multiply (×), or divide (÷) numbers.
⚡ Events — When things happen

Event blocks (with rounded tops) tell your code when to run. You can have multiple event blocks running at the same time!

  • When Start Clicked Runs once when you press the ▶ Play button.
  • When Note Played Runs every time any note is played — on the keyboard or detected by the mic.
  • When Specific Note Runs only when a certain note (C through B) is played.
  • When Pitch Is Runs when the detected pitch goes above or below a frequency (in Hz).
  • Every N Beats Repeats at a regular interval — every 1, 2, 4 beats (or fractions like 0.5). The speed follows your BPM setting.
🎵 Sound Data — Reading music info

These blocks let you read the current state of the music — great for making visuals react to sound!

  • Current Pitch The pitch being played right now, in Hz (0 if silent). Great for controlling size or position with your voice or keyboard.
  • Current Note Name The current note as text, like "C4" or "A#3". Use it in an If block to check what note is being played.
  • Volume Level How loud the sound is, from 0 (silent) to 100 (full volume).
  • Note Is Playing True or false — is a note currently being played? Use inside an If block.
🔄 Control — Loops and decisions
  • Repeat N Times Run the blocks inside it a certain number of times (1–1000). Great for drawing patterns!
  • If Only run the blocks inside if a condition is true — for example, "if note is playing" then draw a circle.
  • Set Variable Save a value with a name so you can use it later. Variables keep their value between frames!
  • Compare / Logic Check if numbers are equal, greater than, or less than. Combine with AND / OR.

🎵 Music Blocks

Music blocks create beats and melodies that play in perfect time using the BPM you set. Put all music blocks inside the 🎵 Music Start block — it's like the "on/off switch" for your song.

🎵 Beat positions use numbers like 1, 1+, 2, 2+, 3, 3+, 4, 4+. The number is the beat, and + means "the and" — halfway between two beats. So 2+ is the "&" in "1 and 2 and 3 and 4"!
🥁 Drums
  • Play Kick The big, boomy bass drum. Put it on beats 1 and 3 for a classic rock feel.
  • Play Snare The snappy cracking drum. Usually on beats 2 and 4.
  • Play Hi-Hat The shimmery cymbal. Try it on every beat or every half beat for energy.
  • Drum Pattern Load a ready-made drum pattern: Rock Beat, Disco, Hip Hop, or Four on Floor.
🎸 Bass
  • Play Bass Note Play a low note (C2 to B2) for a chosen duration at a beat position. Duration choices: whole, half, quarter, eighth, sixteenth notes, or dotted quarter.
  • Bass Pattern A ready-made bass line in any key (C–B). Styles: Root Notes, Walking Bass, Octave Bounce, or Funky.
🎹 Melody
  • Play Melody Note Play a melody note (C4 to E5, including sharps) with a duration and beat position.
  • Play Chord Play three notes at once — choose a root note (C–B) and quality: Major (happy), Minor (moody), or Diminished (tense).
  • Rest A moment of silence for a chosen duration.
🏗️ Song Structure
  • Section A named block of music — Intro, Verse, Chorus, Bridge, or Outro — that repeats for 1, 2, 4, or 8 bars. Organize your song into parts!
  • Repeat Run music blocks multiple times (1–16 times) without having to copy and paste them.
⏱️ Timing
  • Set Tempo Set the song speed in BPM (beats per minute). 60 BPM = slow ballad. 120 BPM = medium. 160 BPM = fast! Also controls the BPM slider in the studio.
  • Music Start The starting hat block — all your music blocks go inside this one.

🔊 Instrument Selector

The dropdown next to the keyboard changes how the piano sounds:

  • Piano A warm, mellow triangle wave sound that fades naturally — like a real piano.
  • Organ A smooth, sustaining sound with chorus, tremolo, and a bit of grit — like a vintage electric organ.
  • Synth A buzzy sawtooth sound with a filter — a classic electronic synthesizer tone.
🎛️ The Loop Pedal remembers which instrument you were using when you recorded each layer! Layer 1 and Layer 2 can use different instruments.
🦶 Sustain Pedal

The sustain pedal keeps notes ringing even after you lift your finger off the key — just like the right pedal on a real piano!

  • Hold Shift Momentary sustain — notes ring as long as you hold Shift.
  • Caps Lock Toggle sustain on or off. The Sustain: ON indicator lights up green when it's active.
🎤 Microphone Mode

Click 🎤 Mic Off to turn on pitch detection from your microphone. Sing or hum and the studio will detect your pitch in real time! The piano key for the detected note lights up in cyan. Your sound data blocks (Current Pitch, Note Is Playing, etc.) will react to whatever you sing.

🎛️ Loop Pedal

The Loop Pedal lets you record yourself playing the keyboard and repeat it automatically! You can build up two layers of playing — like having two musicians playing together.

▶ How to Make a Loop
  1. Press ● Rec [1] (or press 1 on your keyboard). The button blinks red — you're recording! Play some notes on the keyboard.
  2. Press ■ Stop when you're done. The loop length is set and your recording immediately starts playing back on repeat.
  3. Optional: Press ⊕ Overdub [2] (or press 2) while it's playing to record a second layer on top. Play more notes!
  4. Press ■ Stop again to end the overdub. Now both layers play together.
  5. Press ■ Stop [3] (or press 3) at any time to pause playback. Press ▶ Play [3] to resume.
⌨️ Keyboard Shortcuts
1  Record Layer 1
2  Overdub Layer 2
3  Play / Stop
⚠️ These shortcut keys only work when you're not typing in a text box.
📏 Quantize to Bar

When the Quantize to bar checkbox is ticked, the loop length automatically snaps to the nearest complete bar based on the BPM. This makes it easier to keep your loop in sync with the music blocks in the Block Editor!

When unchecked, the loop ends exactly when you press Stop — even if it's mid-bar.

👀 Visualization

The Loop Pedal shows two views of your recording:

  • Piano Roll Time goes left→right. Pitch goes bottom→top. Each note is a colored rectangle. The white line is the playhead moving in real time.
  • Bar Timeline A simpler view — one horizontal track per layer. Note durations appear as colored bars. The white line shows where you are in the loop.

■ Purple = Layer 1    ■ Pink = Layer 2

🗑️ Clear Buttons
  • Clear L1 Erase Layer 1 but keep Layer 2 and the loop playing.
  • Clear L2 Erase Layer 2 but keep Layer 1.
  • Clear All Erase everything and go back to the beginning.
🔗 Works with music blocks! The BPM slider affects both the Block Editor music and the Loop Pedal's Quantize feature. Try running a drum pattern in the Block Editor, then use the Loop Pedal with Quantize on to record a melody that stays in sync. Note: only one can play at a time — starting one pauses the other.

💡 Tips & Tricks

  • 🎨
    Make music visible! Connect sound to shapes using the Map Value block. Try: Map Current Pitch (200–800 Hz) to Draw Circle's size (10–200). Now the circle grows as you sing higher!
  • 🌈
    Animate with Frame Count. Use Frame Count ÷ 60 to get a slowly increasing number. Plug it into rotation or size to make things spin or pulse automatically.
  • 🥁
    Start with a drum pattern. Drag a Drum Pattern block set to "Rock Beat" into a Music Start block and press Play. Then add bass and melody notes on top one at a time.
  • 🔁
    Use the Loop checkbox. The 🔁 Loop checkbox at the top makes your Blockly music repeat forever. Great for trying out different keyboard lines on top of a steady beat.
  • 🎛️
    Layer your loop pedal. Record a simple bass line on Layer 1 (try notes F, J, K on your keyboard). Then overdub a melody on Layer 2. Change instruments between layers for contrast!
  • 📋
    See the code! Click the "See the Code" panel to watch your blocks turn into JavaScript. If you're curious about programming, try reading it — it's the same code that runs in real apps!
  • 💾
    Auto-save. Your block workspace is automatically saved in the browser. When you come back to the studio later, your blocks will still be there!
  • 🎯
    Right-click in the workspace to undo, redo, clean up blocks, or zoom in and out. You can also use the scroll wheel to zoom the block editor.
